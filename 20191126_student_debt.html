  • quarter: lubridate tiene alguna fucin para esto

  • total: total de plata devuelta a las agencias, asumo que la suma de las siguientes columnas o algo as

  • wage_garnishments: pagos recibidos por salarios embargados

Veamos como se comportan estas variable a lo largo del tiempo. Necesito una nueva variable que unifique ao y trimestre.

loans <- loans %>% 
  mutate(date = yq(paste((year+2000), quarter)))

loans_long <- loans %>% 
  pivot_longer(starting:wage_garnishments, names_to = "variable") 

loans_long %>% 
  group_by(date, variable) %>% 
  summarise(total = sum(value, na.rm = TRUE)) %>% 
  ggplot(aes(date, total)) +
  geom_line(aes(color = variable)) +
  scale_color_viridis_d()

Parece que los prestamos que arrancan suman mucho ms que todo el resto. De hecho no estara entendiendo de donde sacan la plata porque el total devuelto no se parece a lo que arranca todos los trimestres. Veamos si al menos total = a todas las otras variables que tienen que ver con pagos.

loans %>% 
  mutate(resto = total - consolidation - rehabilitation - voluntary_payments - voluntary_payments ) %>% 
  group_by(date) %>% 
  summarise(resto = sum(resto, na.rm = TRUE)) %>% 
  ggplot(aes(date, resto)) +
  geom_line()

Lo interesante es que si bien el total no es lo que yo pensaba, el resto se mantine ms o menos constante excepto en el 3er trimestre del 2017 donde baj un montn.
